# Project Blueprint

## Overview

This project is a simple, visually appealing lottery number generator. It provides users with a set of 6 unique random numbers between 1 and 45, displayed in a clean and modern interface. The application is built using fundamental web technologies: HTML, CSS, and JavaScript, with a focus on good design and user experience.

## Features & Design

*   **Lottery Number Generation:** Generates 6 unique random numbers from 1 to 45.
*   **Modern & Clean UI:**
    *   **Layout:** A centered, card-based layout that is easy to view.
    *   **Typography:** Clear, readable fonts.
    *   **Color Scheme:** A pleasant color palette to enhance the visual appeal.
    *   **Animations:** Subtle animations to make the number generation feel more dynamic and engaging.
*   **Responsive Design:** The layout adapts to different screen sizes, ensuring a good experience on both desktop and mobile devices.

## Current Plan

1.  **Create `index.html`:**
    *   Set up the main structure with a container for the lottery numbers and a "Generate" button.
    *   Link to `style.css` and `main.js`.
2.  **Create `style.css`:**
    *   Implement the modern, clean design with the specified layout, colors, and fonts.
    *   Add responsive styles.
    *   Create a reveal animation for the numbers.
3.  **Create `main.js`:**
    *   Implement the logic for generating 6 unique random numbers.
    *   Add an event listener to the "Generate" button.
    *   Display the generated numbers in the UI, triggering the reveal animation.
